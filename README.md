<h1>Calculadora MIPS</h1>

<p>Calculadora desenvolvida em MIPS para a disciplina de Programação em Baixo Nível.</p>

<h2>Operações que são possiveis fazer na calculadora: </h2>

<ol>
	<li>Adição</li>
	<li>Subtração</li>
	<li>Multiplicação</li>
	<li>Divisão</li>
	<li>Potência</li>
	<li>Raiz Quadrada</li>
	<li>Tabuada de um número</li>
</ol>
